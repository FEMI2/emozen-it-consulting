# Emozen IT Consulting

Elite IT consulting web application built with React + TypeScript + Tailwind CSS v4 + Vite.

## Tech Stack

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Motion (formerly Framer Motion)
- **Charts:** Recharts
- **Routing:** React Router v7
- **Build Tool:** Vite
- **Fonts:** Syne (headings) + Inter (body) via Google Fonts

## Project Structure

```
/
├── App.tsx              # RouterProvider root
├── routes.tsx           # React Router config
├── main.tsx             # Entry point
├── index.html           # HTML shell
├── favicon.svg          # Favicon
├── pages/
│   ├── MainSite.tsx     # Main marketing site
│   └── PortalPage.tsx   # Client portal entry
├── components/
│   ├── Navigation.tsx   # Smart-hide nav + mobile menu
│   ├── Hero.tsx         # Full-bleed hero with cinematic zoom
│   ├── TrustBar.tsx     # Scrolling client marquee
│   ├── BentoServices.tsx # 4-service grid
│   ├── StatsSection.tsx # Animated count-up metrics
│   ├── VisionarySection.tsx # Cloud infra section
│   ├── ProblemSolver.tsx    # Cybersecurity section
│   ├── CaseStudies.tsx      # 3 case study cards
│   ├── ManifestoStrip.tsx   # Word-by-word manifesto
│   ├── WhyEmozen.tsx        # Differentiators
│   ├── Testimonials.tsx     # Client quotes
│   ├── Careers.tsx          # Jobs + values
│   ├── CTA.tsx              # Contact form
│   ├── Footer.tsx           # Full footer
│   ├── ScrollProgress.tsx   # Top progress bar
│   ├── AnimatedBackground.tsx # Dot grid background
│   ├── Logo.tsx             # Emozen logo component
│   ├── LoginModal.tsx       # Portal login page
│   └── ClientPortal.tsx     # Full dashboard (8 sections)
└── styles/
    └── globals.css      # Global styles + Tailwind config
```
